SIGSS+ v1.0.0

O SIGSS+ é uma camada inteligente desenvolvida para rodar sobre o sistema SIGSS, adicionando automações e facilidades de cache sem alterar os dados ou o funcionamento original do sistema.

Autor: Guilherme Paicheco Ferreira
Servidor Público • Prefeitura Municipal de Betim
O projeto foi criado voluntariamente com o objetivo de melhorar a produtividade e otimizar o fluxo de trabalho dos servidores da Atenção Primária à Saúde (APS) de Betim.
